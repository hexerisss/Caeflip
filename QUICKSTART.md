# ⚡ Caeflip Quick Start Guide

## 🚀 Start in 3 Steps

### 1️⃣ Install (First Time Only)
```bash
npm install
```

### 2️⃣ Start Backend
Open Terminal 1:
```bash
node server.js
```
✅ Should see: "Connected to Redis" and "API running on http://localhost:3001"

### 3️⃣ Start Frontend
Open Terminal 2:
```bash
npm run dev
```
✅ Should see: "Local: http://localhost:5173"

### 4️⃣ Open Browser
```
http://localhost:5173
```

## 🎲 Play Games

### Crates
1. Go to **Crates** tab
2. Select a case (Starter/Premium/Elite/Godly)
3. Click **Open**
4. Watch spin animation
5. Win item + see in **Inventory**

### Towers
1. Go to **Towers** tab
2. Enter bet amount
3. Choose difficulty (Easy/Medium/Hard)
4. Click **Start Game**
5. Click tiles to climb
6. Reach top = WIN! Hit bomb = LOSE!

### Mines
1. Go to **Mines** tab
2. Enter bet amount
3. Set number of mines (1-10)
4. Click **Start Game**
5. Click safe tiles (avoid mines)
6. **Cashout** anytime or risk for higher multiplier

## ❌ Troubleshooting

| Problem | Solution |
